Primitives

There are seven primitive data types in Java: byte, short, int, long, float, double, boolean, char. They classify information you put into programs into different categories based on their characteristics.

The byte data type are for twoâ€™s compliment integers from -128 to 127. (byte x = -128;)

The short data type deals with the same types of numbers from -32,768 to 32,767. (short x = 32000;)

The int (Integer) data type has to deal with all integers from -231 to 231. (int x = 2;)

The long data type, the largest data type, has to deal with integers from -263 to 263. (long x = 59;)

Floats are numbers from 1.40129846432481707e-45 to 3.40282346638528860e+38 that go to 7 decimal places while doubles go from 4.94065645841246544e-324d and 1.79769313486231570e+308d and 15-16 decimal places, making them twice as specific as floats. (float x = 1.3252453; double y = 3.369048930246853;)

Moving away from numbers, inputs of either true or false are classified into the boolean data type and chars (Characters) are are single 16-bit unicode characters, from 0 to 65,535 and a to z. (boolean x = true; char y = "A";)
