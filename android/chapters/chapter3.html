Loops

Loops are useful when you want to complete a repeated set of instructions over and over. There are two main types of loops in Java: for loops and while loops.

For loops are generally used when the number of iterations (repetitions) are known before the execution of the loop The format for writing a for loop is :

for(int i = 0; i < (number of repetitions you want); i++) {
[code]
}

While loops are used when the number of iterations is unknown. Each has their own syntax for writing, and each has its own situation that theyâ€™re best used in. The format for writing a while loop is :

while(boolean) {
[code]
}

For example, if you want to create a line of robots and also store them into an array (more on arrays later), a for loop would be the best choice. If you have an array with an unknown amount of integers and you want to print each one, a while loop would be best. Although the uses for each loop are different, they both check to see if a certain condition is met to decide whether or not to continue iterating.
