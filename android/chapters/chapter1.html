Basic Syntax and Java

Java is an object-oriented programming language. This means that any code is directed towards objects that you create. You control these objects using sets of instructions called methods, and what you can do with them is limitless. Overall, the Java syntax is very simple to learn and to use.

The main thing to remember is that for most statements that you will make in Java you will need a semicolon at the end of it. The semicolon signals Java that you are done with that particular segment of code.

The next most important thing to remember is that any block of code that has been declared has to be in curly braces ( { } ) these signify the start and end of the declared code.

Two statements that should be known are the System.out.print() and System.out.println() commands. System.out.print() prints out whatever is in the paretheses, but System.out.println() prints out whatever is in the parentheses but also moves down a line.

The final thing you need to keep in mind is that any parameters you have for any method go into parentheses.
