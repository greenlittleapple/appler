Arrays

Arrays are a great way to store large amounts of a single type of information. They are similar to dictionaries in that they store data and then that information can be accessed by going to a certain location. They also are similar due to the fact that they both only store one type of data, as dictionaries hold words while arrays can hold ints, doubles, etc. Arrays eliminate the need to declare individual variables for every single bit of information you want to store, as you can just put all of it into one place and then you can easily access them. Just as dictionaries store information in pages, arrays keep their data in indices (index for singular). Keep in mind that the index numbers start with zero being the first index, one being the second, etc. Creating a reference to an array is as simple as the following:
Datatype [] name;
 Datatype refers to what type of information is going to be stored, ie ints, floats, etc. The [] is there to signify that itâ€™s an array and the name is simply the name of the array. To construct an array:
new datatype [x];
 New simply declares your intention to make a new array, the datatype should match the previous one in the reference, and x represents how many values are going to be in your array. To put data into the array, all you have to do is to set the value of an array at a certain index to a certain value that matches the datatype.
Name [ x ] = value;
 Name is the name of the array, x is the index and value is whatever you want to set to that index. To access data at a point in an array, you just call information from the array.
Name [ x ];
 Name is the name of the array you want to access and x is the index at which you want to pull data from.
