Strings

Strings in Java are just a collection of characters. They’re treated as objects, so they have their own set of methods that accompany them that do everything from cut out parts of it to adding strings together. You declare strings as you would any other type of variable, for example:
String words = “Hello World!”;
 Notice the capitalized “S” on the word “String” as the data type. Also remember to surround any of your strings with double quotations “”. One method that the String class has is the length() method. This returns an int with the number of characters in the string. Spaces (“ “) and symbols (!, @, #) also count as characters, so for example in the statement above “Hello World!” would have 12 characters. Strings can contain numbers, but usually numbers are stored as their respective types (int, double, etc).
