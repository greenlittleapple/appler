1 - Basic Syntax




To make an Android app, you must learn the basics and symbols of Java. In this chapter, we will go over the mandatory syntax and vocabulary in Java.

2 - Primitives




Primitives are useful types of variables. Examples of Primitives are Integers, Doubles (Integers that can have a decimal point), Characters, and Booleans.

3 - Loops







Loops help repeat actions both definitely and indefinitely. Loops rely on Booleans and Integers, which are Primitives.

4 - Recursion




Recursion is another type of way to repeat actions but it can often solve problems that loops cannot. We will also learn about Stacks in this Chapter.

5 - Arrays







Arrays store data in indexed lists of single types. Using arrays allows a simpler way of storing variables and integers for use later.
