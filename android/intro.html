INTRODUCTION TO JAVA

We will go over and learn the basics of Java.

Welcome to the first step in your journey in becoming an Android app designer! Here you will learn the basics of Java and programming. Android apps use the programming language called Java to run their code.

Preparation

1. Download and extract Eclipse SDK, a programming interface for Java and Android apps. Download the 32-bit or 64-bit package.

2. Run "SDK Manager" in the folder that you extracted and check "Tools" at the top and "Android 5.0 (API 21)", and click "Install packages" at the bottom.

3. Run "eclipse.exe" in the "eclipse" folder and at the top, click Help > Install New Software.

3b. Click "Add" on the top right and type in "ADT Plugin" for the Name and "https://dl-ssl.google.com/android/eclipse/" into the Location and click "OK".

3c. In the next dialog, check "Developer Tools" and click "Next", then "Next" again, and finally accept the agreements and click "Finish" and let it install.

3d. Restart Eclipse when prompted.

4. Go to File > New on the top left and verify that "Android Application Project" appears.

5. Click "Java Project", put in a Project Name, and click Finish. On the left, an icon followed by your Project Name should appear. Right click it, hover over "New", and click "Class" on the right. Name the class.

6. Type in the line between the 2 brackets :
public static void main(String[] args) {

}

 7. You are now ready to learn Java. Head on to the Chapters -->
